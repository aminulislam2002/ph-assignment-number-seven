Question 01: What is the deference between .map(), .filter(), .forEach() and .find()?

Answer:

.map() methods is used to executes the same code on every element in an array and returns a new array with the selected updated elements.

.filter() methods is used to checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.

.forEach() methods is used to execute the same code on every element in an array but does not change the array and it will not returns anything

.find() is used to returns the first element in the provided array that satisfies the provided testing function. If no values satisfy the testing function, undefined is returned.

Question 02: What is the deference between Local Storage, Session Storage and Cookies?

Answer:

Local Storage: It is a web storage method that helps us store data without any expiration date. Saving data is not deleted when the browser is closed and data is available for future sessions. Stored data in key\value pairs in the browser.

Session Storage: It is a web storage method that helps us stores in browser for a session. Saving data is deleted when the browser is closed and data is not available for future sessions in key\value pairs in the browser.

Cookie: It is a web storage method that data stored in small text file. Naturally sent by visited websites. Sent to users computer through browser. Is not a program, dosen't perform any function. Stored data in key\value pairs in the browser.
